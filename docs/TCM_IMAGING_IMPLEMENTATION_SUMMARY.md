# 影像 - 中医病机分析模块实施总结

## 项目概述

**目标**：在乳腺超声影像分析中融入中医理论，提供病机倾向参考

**定位**：研究性质功能，辅助参考，不能替代四诊合参

**实施时间**：2026-05-29 ~ 2026-05-30

**状态**：Phase 1-2 完成 ✅ (后端开发完成，等待数据库就绪后测试)

---

## 实施成果

### 1. 理论框架设计

#### 核心理念
- 将超声影像特征作为中医"望诊"的延伸
- 基于循证医学文献（B/C 级证据）
- 输出病机"倾向"而非"诊断"
- 明确标注证据等级和局限性

#### 病机维度（4 个）
1. **瘀血 (stasis)** - 血液运行不畅
2. **痰浊 (phlegm)** - 水液代谢障碍
3. **毒邪 (toxin)** - 致病力强的邪气
4. **正虚 (deficiency)** - 正气不足

### 2. 规则引擎实现

#### 影像特征（20+ 项）

**形态学特征（8 项）**
- 边界：清晰/不清/毛刺/角征
- 形态：规则/不规则/蟹足样
- 纵横比：<1 / >1
- 边缘：光整/模糊/微分叶/宏分叶
- 包膜：完整/不完整/无

**内部回声特征（6 项）**
- 回声类型：无回声/低回声/等回声/高回声
- 均匀度：均匀/不均匀
- 钙化：无/粗大/点状微钙化/簇状/弧形
- 液化坏死：有/无

**血流动力学特征（4 项）**
- CDFI 分级：0 级/I 级/II 级/III 级
- 血流分布：周边型/穿入型/网状型
- RI 阻力指数：<0.7 / >0.7

**弹性成像特征（2 项）**
- 弹性评分：1-5 分
- 硬度比：<2.5 / >4.5

**生长动力学特征（3 项）**
- 生长速度：稳定/缓慢/中等/快速
- 大小：<1cm / 1-2cm / >3cm
- 多灶性：有/无

#### 权重计算

```python
score = Σ(feature_weight × evidence_modifier)

evidence_modifier = {
    "A": 1.0,   # 多中心 RCT（暂无）
    "B": 0.8,   # 单中心对照研究
    "C": 0.6,   # 观察性研究/专家共识
}

tendency_score = min(1.0, base_score / max_possible_score)
```

## 创新点

### 1. 循证医学基础

- 基于 7 篇中英文文献
- 明确标注证据等级（A/B/C）
- 不同于简单查表映射

### 2. 定量倾向分析

- 0-1 连续评分
- 四维度并行评估
- 可视化展示

### 3. 负责任的设计

- 明确"研究性质"
- 多次声明"仅供参考"
- 强调"四诊合参"必要性

### 4. 可解释性强

- 每项评分都有证据摘要
- 规则透明，可追溯
- 医师可审核修正

---

## 局限性与改进方向

### 当前局限

1. **证据等级有限** - 主要为 B/C 级，缺乏 A 级 RCT 证据
2. **规则基于文献** - 未经大规模临床验证
3. **特征提取依赖** - 需要结构化报告或 AI 模型
4. **数据库依赖** - 当前数据库表未创建（等待 PostgreSQL 就绪）

### 改进方向

1. **权重校准** - 收集 300+ 病例，与专家辨证对比
2. **规则优化** - 根据一致性分析调整权重
3. **特征扩展** - 纳入更多影像组学特征
4. **AI 融合** - 深度学习模型学习影像-病机关联
