# 医疗 Copilot 系统架构文档

## 概述

医疗 Copilot 是一个**贯穿整个项目的智能聊天助手**，通过自然语言对话协助医生完成诊疗全流程工作。

## 系统定位

## 核心能力

### 1. 影像分析解释 (Imaging Mode)

**功能**：
- 解释 AI 超声分析结果（BI-RADS 分级、肿块特征）
- 说明影像处理流程和技术细节
- 对比历史影像变化
- 回答影像质量相关问题

**对话示例**：
```
医生：3 号患者的 AI 分析结果怎么看？

Copilot：患者张三（ID: 3）的超声分析结果如下：

📊 **AI 分析摘要**
- BI-RADS 分级：4a 类（低度可疑）
- 肿块位置：右乳外上象限，1 点钟方向
- 大小：12.3 × 8.7 × 10.2 mm
- 形态：不规则
- 边界：模糊，有毛刺
- 回声：低回声
- 钙化：点状钙化
- 血流：CDFT 可见少量血流信号

💡 **临床建议**
1. 建议穿刺活检明确诊断
2. 与 3 个月前影像对比，肿块增大 2mm
3. 形态和边界特征提示恶性风险

需要我详细解释某个特征吗？
```

### 2. 诊断辅助 (Diagnosis Mode)

**功能**：
- 解释中西医诊断依据
- 提供辨证分型建议
- 分析鉴别诊断
- 推荐治疗方案

**对话示例**：
```
医生：这个患者的中医辨证是什么？

Copilot：根据患者信息，中医辨证分析如下：

📋 **四诊信息**
- 症状：乳房胀痛，经前加重，心烦易怒
- 舌象：舌红，苔薄黄
- 脉象：弦细数

🔍 **辨证分析**
- 病位：肝经所过之乳房
- 病性：肝郁气滞，郁久化热
- 证型：**肝郁气滞证**

💊 **治则治法**
- 治法：疏肝理气，清热散结
- 方剂：逍遥散加减
- 中药：柴胡 10g, 白芍 15g, 当归 10g, 茯苓 15g, 
        白术 10g, 薄荷 6g, 夏枯草 15g, 浙贝母 10g

需要我生成完整的中医诊疗方案吗？
```

### 3. 系统控制 (Control Mode)

**功能**：
- 创建/更新患者档案
- 安排超声检查
- 生成检查报告
- 导出数据统计
- 安排随访计划

**对话示例**：
```
医生：给李四安排下周的复查

Copilot：好的，我来帮您安排随访。

📅 **随访安排**
- 患者：李四（ID: 12）
- 类型：超声复查
- 建议时间：2026-06-05（下周二）上午
- 注意事项：月经结束后 3-7 天最佳

确认要创建这个随访计划吗？

[确认] [取消] [修改时间]
```

```
医生：生成今天的检查报告

Copilot：今天共完成 15 例超声检查，其中：

📊 **报告生成统计**
- 已完成：12 份
- 待审核：3 份
- 待生成：0 份

待审核报告：
1. 王五 - BI-RADS 3 类
2. 赵六 - BI-RADS 4a 类
3. 钱七 - BI-RADS 2 类

要批量审核并生成报告吗？

[批量生成] [逐一审核] [取消]
```

### 4. 通用对话 (General Mode)

**功能**：
- 回答医疗知识问题
- 查询系统功能说明
- 提供操作指导
- 日常交流

### 工作模式

| 模式 | 触发关键词 | 功能 | 颜色主题 |
|------|-----------|------|---------|
| **general** (通用对话) | 默认 | 回答各类问题 | 紫色 (#667eea) |
| **imaging** (影像分析) | 影像、超声、BI-RADS、肿块 | 解释 AI 分析结果 | 绿色 (#11998e) |
| **diagnosis** (诊断辅助) | 诊断、辨证、证型、舌苔 | 中西医疗诊断建议 | 粉色 (#f093fb) |
| **control** (系统控制) | 创建、删除、导出、生成 | 执行系统操作 | 橙色 (#fa709a) |

## API 端点

### 创建会话
```http
POST /api/v1/copilot/sessions
Content-Type: application/json

{
  "user_id": 1,
  "patient_id": 12,
  "visit_id": 34,
  "mode": "general"
}
```

### 发送消息
```http
POST /api/v1/copilot/sessions/{session_id}/messages
Content-Type: application/json

{
  "message": "查看 3 号患者的影像分析结果"
}

Response:
{
  "session_id": "uuid",
  "response": "好的，正在为您查询...",
  "mode": "imaging",
  "suggested_actions": ["查看详细报告", "对比历史影像", "生成诊断建议"],
  "updated_at": "2026-05-29T10:00:00"
}
```

### 执行系统命令
```http
POST /api/v1/copilot/sessions/{session_id}/commands
Content-Type: application/json

{
  "command": "generate_report",
  "params": {
    "patient_id": 12,
    "visit_id": 34
  }
}
```

### 交互特性

1. **自动模式切换** - 根据对话内容自动切换到对应模式
2. **建议操作** - 动态生成相关操作建议
3. **快捷输入** - 点击建议操作快速输入
4. **打字机动画** - AI 回复时显示打字动画
5. **消息历史** - 保存完整对话历史
6. **响应式设计** - 自适应不同屏幕尺寸

## 数据模型

### ChatSession
```python
class ChatSession(BaseModel):
    session_id: str
    user_id: Optional[int]  # 医生 ID
    patient_id: Optional[int]  # 关联患者
    visit_id: Optional[int]  # 关联就诊
    messages: List[ChatMessage]
    mode: str  # general/imaging/diagnosis/control
    status: str  # active/paused/closed
    context: Dict[str, Any]  # 上下文信息
```

### ChatMessage
```python
class ChatMessage(BaseModel):
    role: str  # user/assistant/system
    content: str
    timestamp: str
    metadata: Dict[str, Any]
```

## 安全与隐私

### 安全措施

1. **权限验证** - 只有认证医生可访问
2. **患者隐私** - 自动脱敏患者信息
3. **操作审计** - 记录所有系统操作
4. **命令白名单** - 仅允许预定义的命令

### 安全边界

- ❌ 不替代医生做出最终诊断
- ❌ 不开具处方（仅提供建议）
- ❌ 不处理紧急情况（建议立即就医）
- ❌ 不提供与诊疗无关的建议

## 部署架构

### Docker 容器化

```yaml

services:
  copilot-service:
    build: ./backend
    ports:
      - "8000:8000"
    environment:
      - MODEL_PATH=/models/medical-copilot
      - GPU_ENABLED=true
    volumes:
      - ./models:/models
      - ./data:/data
    deploy:
      resources:
        reservations:
          devices:
            - driver: nvidia
              count: 1
              capabilities: [gpu]

  frontend:
    build: ./frontend
    ports:
      - "3000:3000"
    depends_on:
      - copilot-service
```

```
- 肿块大小：12.3 × 8.7 × 10.2 mm
```
