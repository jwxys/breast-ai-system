**输入数据仅有**：
- ✅ 超声影像（BI-RADS 分级、肿块特征）
- ✅ 基本信息（年龄、性别、乳腺状况）

**但系统设计了**：

## 重要说明

### 为什么移除？

1. **无证型依据** - 中医辨证需要舌象、脉象、详细问诊
2. **误导医生** - 显示无依据的诊断结果
3. **临床风险** - 可能导致错误治疗
4. **法律风险** - 提供虚假医疗信息
5. **专业声誉** - 降低系统可信度

### 中医辨证需要什么？

```
✅ 望诊：舌质、舌苔、面色、形体
✅ 闻诊：声音、气味
✅ 问诊：十问歌、症状清单、月经生育史
✅ 切诊：脉象（弦/滑/涩/细等）

❌ 当前系统：仅有超声影像 + 基本信息
```

### 未来如需中医功能

必须：
1. 补全四诊信息采集表单
2. 采用标准化体质评估量表
3. 聘请中医专家顾问
4. 进行临床验证
5. 按医疗器械注册
