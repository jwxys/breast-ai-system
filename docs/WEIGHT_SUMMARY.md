## 西医分支权重

### PBS-Net 病灶分割
```
权重文件：models/pbs_net_v12.pth (128 MB)
训练数据：2,500 例本院超声 (双医师标注 + 仲裁)
性能指标：Dice 0.87, IoU 0.78, Hd95 3.2mm
伦理审批：IRB-2023-BREAST-001
```

### DFMFI 特征融合
```
权重文件：models/dfmfi_v20.pth (96 MB)
训练数据：3,000 例多模态 (B 模式 + 多普勒 + 弹性)
性能指标：Acc 0.94, AUC 0.97, 参数量 12.8M
骨干网络：ResNet-34 改进版 + Transformer 注意力
```

### HXM-Net 多模态
```
权重文件：models/hxm_net_v15.pth (256 MB)
训练数据：1,500 例 (超声 + 钼靶+MRI)
性能指标：Acc 0.94, AUC 0.96, Kappa 0.89
模态权重：B 超 0.45, 多普勒 0.30, 弹性 0.25
```

### BI-RADS 规则引擎
```
权重文件：无 (规则引擎)
指南来源：ACR BI-RADS Atlas 2015 版
特征评估：形状/边缘/方位/回声/后方回声
分类逻辑：0-6 级标准化评估
```

## 中医分支权重

### 体质辨识 (TCM-CIN)
```
权重文件：models/tcm_constitution_v31.pth (64 MB)
训练数据：5,000 例体质问卷 (3 家三甲医院)
量表来源：《中医体质分类与判定自测表》改良版
性能指标：Acc 0.89, Macro-F1 0.87, Kappa 0.85
体质分布：9 种体质 (痰湿 15%/血瘀 13%/气虚 12%...)
```

### 证型识别 (TCM-SDN)
```
权重文件：models/tcm_syndrome_v23.pth (72 MB)
训练数据：3,200 例临床病例 (本院 + 协作医院)
证型标准：《中医病证诊断疗效标准》ZY/T001.1-94
性能指标：Acc 0.86, 主证识别 0.91, 兼证识别 0.78
6 大证型：肝郁气滞 22%/冲任失调 20%/痰凝气滞 18%...
```

### 方剂推荐 (TCM-PKG)
```
数据文件：models/tcm_knowledge_graph.json (2.4 MB)
知识来源：12 部经典 + 3 部指南 + 48 首经验方 + 156 篇文献
方剂规模：36 首基础方，280 种加减变化
推荐算法：证型×0.5 + 症状×0.3 + 体质×0.2 - 禁忌扣分
```
