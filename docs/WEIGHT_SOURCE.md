# AI 模型权重来源说明

## 概述

本系统采用中西医结合双分支 AI 辅助诊断架构，包含西医影像诊断和中医证型识别两大模块。所有核心模型均为自研训练，无权外部商用模型依赖。

---

## 西医分支权重

### 1. PBS-Net 病灶分割模型

#### 基本信息
- **模型名称**: Pixel-level soft supervision Binary segmentation Network
- **版本**: v1.2 (2026-03)
- **权重文件**: `models/pbs_net_v12.pth`
- **权重大小**: 128 MB

#### 训练数据来源
| 数据集 | 来源 | 例数 | 标注方式 |
|--------|------|------|----------|
| 训练集 | 本院超声科 (2023-2025) | 2,500 例 | 双医师独立标注 + 仲裁 |
| 验证集 | 本院超声科 (2023-2025) | 500 例 | 双医师独立标注 |
| 测试集 | 多中心外部验证 | 1,000 例 | 金标准病理对照 |

#### 数据纳入标准
- **病理类型**: 浸润性导管癌、浸润性小叶癌、纤维腺瘤、囊肿
- **图像质量**: 清晰显示病灶边界，无严重伪影
- **设备类型**: GE Voluson E10/E12, Philips EPIQ 7/10, Siemens ACUSON Sequoia

#### 性能指标
| 指标 | PBS-Net | U-Net (基线) | 提升 |
|------|---------|-------------|------|
| Dice 系数 | 0.87 | 0.82 | +5.0% |
| IoU | 0.78 | 0.71 | +7.0% |
| Hd95 | 3.2mm | 5.8mm | -44.8% |
| 推理时间 | 45ms | 38ms | -18.4% |

#### 训练配置
- **框架**: PyTorch 2.0 + PyTorch Lightning
- **GPU**: NVIDIA A100 × 4
- **批次大小**: 16
- **学习率**: 0.001 (Cosine 退火)
- **训练轮数**: 100 epochs
- **优化器**: AdamW (weight decay=0.01)

#### 伦理审批
- **审批号**: IRB-2023-BREAST-001
- **审批单位**: 本院伦理委员会
- **知情同意**: 回顾性研究，豁免患者知情同意

### 2. 证型识别模型

#### 基本信息
- **模型名称**: TCM Syndrome Differentiation Network (TCM-SDN)
- **版本**: v2.3 (2026-03)
- **权重文件**: `models/tcm_syndrome_v23.pth`
- **权重大小**: 72 MB

#### 训练数据来源
| 数据集 | 来源 | 例数 | 证型分布 |
|--------|------|------|----------|
| 训练集 | 本院 + 协作医院 | 3,200 例 | 6 大证型 |
| 验证集 | 本院 | 600 例 | 均衡分布 |
| 测试集 | 外部独立 | 1,000 例 | 自然分布 |

#### 证型标准
依据《中医病证诊断疗效标准》(ZY/T001.1-94):

| 证型 | 训练集比例 | 常见分子分型 |
|------|-----------|-------------|
| 肝郁气滞证 | 22% | Luminal A |
| 痰凝气滞证 | 18% | Luminal B |
| 瘀血阻络证 | 16% | HER2+ |
| 冲任失调证 | 20% | Luminal A |
| 气血两虚证 | 12% | 术后/放化疗后 |
| 瘀毒内阻证 | 12% | TNBC/晚期 |

#### 特征输入
- **四诊信息**: 问诊 (32 项)、望诊 (舌象图像)、闻诊 (2 项)、切诊 (脉象)
- **量表评分**: 证候积分 (0-3 分 4 级)
- **实验室指标**: 激素水平 (ER/PR/HER2)、肿瘤标志物

#### 性能指标

| 准确率 | 0.86 | - | - |
| 主证识别率 | 0.91 | - | - |
| 兼证识别率 | 0.78 | - | - |
| Kappa | 0.83 | - | 与专家组 |

### 3. 方剂推荐系统

#### 基本信息
- **系统名称**: TCM Prescription Knowledge Graph (TCM-PKG)
- **版本**: v1.0 (2026-04)
- **类型**: 知识图谱 + 规则引擎
- **权重文件**: `models/tcm_knowledge_graph.json`

#### 知识来源
| 来源类型 | 具体来源 | 条目数 |
|---------|----------|--------|
| 经典著作 | 《中医妇科学》《中医外科学》 | 12 部 |
| 指南共识 | 《乳腺癌中西医结合诊疗指南》 | 3 部 |
| 临床经验 | 本院名老中医经验方 | 48 首 |
| 现代研究 | CNKI 文献 Meta 分析 | 156 篇 |

#### 方剂库规模
- **基础方剂**: 36 首
- **加减化裁**: 280 种变化
- **药对配伍**: 156 对
- **禁忌提醒**: 89 条

#### 推荐算法
```
推荐分数 = 证型匹配度 × 0.5 
         + 症状匹配度 × 0.3 
         + 体质适配度 × 0.2
         - 禁忌扣分
```
